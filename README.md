Real time chat application built using Spring Boot WebSocket and Thymeleaf
This project demonstrates how to create a simple messaging system where multiple users can connect and communicate instantly through a web interface

Features
Real time messaging using WebSocket
Simple and clean user interface with Thymeleaf
Spring Boot backend for handling connections and messages
Broadcast messages to all connected users
Lightweight and easy to understand project structure

Tech Stack
Java
Spring Boot
WebSocket
Thymeleaf
HTML CSS Bootstrap

Purpose
This project is built to understand real time communication in web applications and to learn how Spring Boot handles WebSocket connections

How to Run
Clone the repository
Open in IntelliJ or any IDE
Run the Spring Boot application
Open browser and go to localhost port 8080

Future Improvements
Private messaging between users
User authentication and login system
Message history storage using database
Typing indicators and online status
